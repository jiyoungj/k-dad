Database Description

1. Data collection

This section describes the recording procedures, experimental design, and equipment used during the data collection process.

1.1. Devices used

Articulatory data: Carstens AG501
Acoustic data: Microphone & TASCAM

1.2. Experiment setup

• Sound-attenuated room in Hanyang Institute for Phonetics and Cognitive Sciences of Language
• TASCAM HC-P2 digital recorder and a SHURE KSN44 condenser microphone
• Eight sensor coils attached (tongue tip, body, dorsum, lower/upper incisors, lower/upper lips, nasion)

3. Data format

The database includes:

  • Audio recordings (WAV format)
  • Articulatory data (MAT format)
  • Annotated TextGrid files
  • Accompanying metadata files for each speaker (PDF format)
